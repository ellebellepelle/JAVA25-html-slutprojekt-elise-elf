# 🌐 JAVA25-html-slutprojekt-elise-elf

A personal and responsive multi-page website built with HTML, CSS and JavaScript.

## 🌍 Live Demo
https://ellebellepelle.github.io/JAVA25-html-slutprojekt-elise-elf/

## 📌 Description
This project is a personal website with multiple pages, a navigation bar, image galleries, interactive forms and some Bootstrap components.

## ✨ Features
- Multi-page website
- Responsive layout
- Navigation bar
- Image gallery
- Clickable images with JavaScript
- Interactive contact forms
- Bootstrap Accordion
- Bootstrap Modal
- Bootstrap Alert

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript
- Bootstrap
- GitHub Pages

## 🚀 How to run locally
1. Clone the repository:
   ```bash
   git clone https://github.com/ellebellepelle/JAVA25-html-slutprojekt-elise-elf.git
   ```
2. Open the project in IntelliJ IDEA or another code editor
3. Open `index.html` in your browser

## ⚠️ Notes
- This is a static website hosted with GitHub Pages
- Some interactive parts are handled with JavaScript
